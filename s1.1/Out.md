এরপর মুঘল সম্রাট ঔরঙ্গজেবের শাসন ছিল সবচেয়ে বিতর্কিত। তিনি ছিলেন একজন कट्टर ইসলাম এবং তাঁর অনেক সংস্কার মুঘল সমাজ ও অর্থনীতির ওপর খারাপ প্রভাব ফেলেছিল। তিনি রাস্তাঘাট ধ্বংস করে দিয়েছিলেন, যাতে খ্রিস্টান বণিকরা প্রবেশ করতে না পারে। তিনি সৈন্যদের বেতন দিতেও অনিচ্ছুক ছিলেন, যার ফলে সাম্রাজ্যে বিদ্রোহের আগুন জ্বলতে থাকে। ঔরঙ্গজেবের যুদ্ধগুলো সাম্রাজ্যের অর্থনীতিকে আরও দুর্বল করে দেয়।

শুধু রাজনৈতিক দুর্বলতাই নয়, মুঘল সাম্রাজ্যের পতনের পেছনে আরও কিছু কারণ ছিল:

*   অর্থনৈতিক সংকট: সাম্রাজ্যের রাজস্ব ব্যবস্থা ভেঙে পড়েছিল। অতিরিক্ত করের বোঝা সাধারণ মানুষের ওপর চাপায়, যার ফলে বিদ্রোহ দেখা যায়।
*   সামরিক দুর্বলতা: মুঘল সেনাবাহিনী দুর্বল হয়ে পড়েছিল। দাহিয়নেরা এবং অন্যান্য আঞ্চলিক শক্তির উত্থান মুঘলদের জন্য একটা বড় চ্যালেঞ্জ হয়ে দাঁড়ায়।
*   ভাষা ও সংস্কৃতির প্রভাব: বিভিন্ন জাতি ও ধর্মের মানুষের মধ্যে মিশ্রণ ঘটে। ফারসি ভাষার প্রভাব বাড়তে থাকে এবং মুঘল সংস্কৃতি তার ঐতিহ্য হারাতে থাকে।
*   ইউরোপীয় বণিকদের প্রভাব: ইউরোপীয় বণিকরা ভারতের অর্থনীতিতে প্রবেশ করে এবং তারা মুঘলদের সাথে বাণিজ্য করে নিজেদের প্রভাব বিস্তার করতে থাকে।

এসব কারণ combined হয়ে মুঘল সাম্রাজ্যের পতন ডেকে আনে। এটা একটা দীর্ঘ প্রক্রিয়া ছিল, যার শুরু হয়েছিল দুর্বল শাসন থেকে এবং শেষ হয়েছিল রাজনৈতিক, অর্থনৈতিক ও সামরিক দুর্বলতার কারণে।

৩. একাধিক ইনপুট — বহু ভ্যারিয়েবল
============================================================
কোড রিভিউ: কোডটি খুবই ছোট এবং সরল। তবে একজন সিনিয়র পাইথন ডেভেলপার হিসেবে আমি এর কিছু দিক নিয়ে আলোচনা করব:

**কোডের ভালো দিক:**

*   **সরলতা:** কোডটি খুবই সহজে বোঝা যায় এবং এর কাজও স্পষ্ট।
*   **কার্যকারিতা:** `add` ফাংশনটি দুটি সংখ্যার যোগফল সঠিকভাবে প্রদান করে।

**সম্ভাব্য উন্নতির ক্ষেত্র:**

*   **Naming Convention:**
    *   ফাংশনের নাম `add` খুবই ভালো। এটি ফাংশনের কাজ স্পষ্টভাবে বোঝায়।
    *   ভেরিয়েবলের নাম `a` এবং `b` সাধারণত সংখ্যার জন্য ব্যবহৃত হয়। তবে, আরও স্পষ্টতার জন্য `num1` এবং `num2` অথবা `x` এবং `y` ব্যবহার করা যেতে পারে। যদিও এই ক্ষেত্রে naming convention খুব গুরুত্বপূর্ণ নয়, তবুও ভালো অভ্যাস অনুসরণ করা উচিত।

*   **Type Hints:**
    *   পাইথনে Type Hints ব্যবহার করা কোডকে আরও সুস্পষ্ট এবং ত্রুটিমুক্ত করতে সাহায্য করে। এই কোডে Type Hints যোগ করলে এটি আরও শক্তিশালী হবে।
    ```python
    def add(a: int, b: int) -> int:
        return a + b
    ```
    এখানে:
    *   `a: int` এবং `b: int` মানে `a` এবং `b` উভয়ই ইন্টিজার টাইপের হতে হবে।
    *   `-> int` মানে ফাংশনটি একটি ইন্টিজার রিটার্ন করবে।

*   **Error Handling:**
    *   এই কোডে কোনো Error Handling নেই। যদি `a` অথবা `b` এর মান ইন্টিজার না হয়, তাহলে `TypeError` হতে পারে। এই ধরনের ত্রুটিগুলি পরিচালনা করার জন্য `try-except` ব্লক ব্যবহার করা যেতে পারে।
    ```python
    def add(a, b):
        try:
            return a + b
        except TypeError:
            raise TypeError("Arguments must be numbers")
    ```

*   **Docstring:**
    *   ফাংশনের শুরুতে একটি Docstring যোগ করা উচিত। Docstring-এ ফাংশনের কাজ, প্যারামিটার এবং রিটার্ন ভ্যালু সম্পর্কে বর্ণনা দেওয়া উচিত।
    ```python
    def add(a: int, b: int) -> int:
        """Returns the sum of two numbers.

        Args:
            a: The first number.
            b: The second number.

        Returns:
            The sum of a and b.

        Raises:
            TypeError: If either argument is not a number.
        """
        try:
            return a + b
        except TypeError:
            raise TypeError("Arguments must be numbers")
    ```

**চূড়ান্ত কোড (Type Hints এবং Docstring সহ):**

```python
def add(a: int, b: int) -> int:
    """Returns the sum of two numbers.

    Args:
        a: The first number.
        b: The second number.

    Returns:
        The sum of a and b.

    Raises:
        TypeError: If either argument is not a number.
    """
    try:
        return a + b
    except TypeError:
        raise TypeError("Arguments must be numbers")
```

এই রিভিউটি আপনার কোডকে আরও উন্নত করতে সাহায্য করবে।  মনে রাখবেন, কোড রিভিউ একটি চলমান প্রক্রিয়া এবং এটি আপনার কোডের গুণগত মান উন্নত করতে সহায়ক।

• আলোর গতি: আলোর গতি হলো শূন্যস্থানে আলোর যে বেগ, সেটি প্রতি সেকেন্ডে ২৯৯,৭৯২,৪৫৮ মিটার (প্রায়)। এই গতিতে আলোকরশ্মি এক স্থান থেকে অন্য স্থানে যেতে পারে। 

বিষয়টি আরো একটু সহজভাবে ব্যাখ্যা করা হলো:

*   আলোর উৎস (যেমন সূর্য বা একটি টর্চলাইট) থেকে আলোকরশ্মি যখন কোনো বস্তুর ওপর পড়ে এবং ফিরে আসে, তখন সেটি একটি নির্দিষ্ট সময় অতিক্রম করে। 
*   আলোর গতি এই সময় পরিমাপের একক হলো প্রতি সেকেন্ডে ২৯৯,৭৯২,৪৫৮ মিটার। 
*   আলোর এই গতিবেগ ধ্রুবক, অর্থাৎ এটি স্থান বা মাধ্যমের ওপর নির্ভর করে না।

আলোর গতির ধারণা পদার্থবিদ্যা এবং মহাবিশ্বের অনেক বিষয় বুঝতে আমাদের সাহায্য করে।
• মাধ্যাকর্ষণ: মাধ্যাকর্ষণ হলো একটি প্রাকৃতিক শক্তি যা মহাবিশ্বের যেকোনো দুটি বস্তুকে একে অপরের দিকে আকর্ষণ করে, যাদের মধ্যে ভর আছে।
• পরমাণু: পরমাণু হলো মৌলিক পদার্থের ক্ষুদ্রতম অংশ যা পদার্থের রাসায়নিক বৈশিষ্ট্য নির্ধারণ করে।
